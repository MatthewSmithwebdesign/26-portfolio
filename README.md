# Matthew Smith Portfolio

A terminal-themed developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Terminal UI with command navigation
- Projects showcase
- Skills display
- Blog with MDX content
- Fully responsive
- Accessibility focused

## Terminal Commands

- `help` - Show available commands
- `about` - Learn about me
- `projects` - View my projects
- `skills` - See my technical skills
- `blog` - Read my blog posts
- `contact` - Get contact information
- `ls` - List available pages

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Adding Content

- **Blog posts**: Add `.mdx` files to `content/blog/`
- **Projects**: Edit `src/app/projects/page.tsx`

## Deploy

Push to GitHub and import in Vercel.
